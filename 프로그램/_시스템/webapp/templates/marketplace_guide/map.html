📖 데이터 코드 지도 — 판매처

각 마켓의 개발환경 + 프로그램과 주고받는 API 전부를 영역별로. 색은 상태에만(됨·연결됨·미정의), 방향은 보내기/받기 위치로. 별(☆)로 담으면 「주요 데이터」에서 마켓끼리 비교.
프로그램 · 가격·재고→ 전송 대기 · 스냅샷→ 판매처 API · 스스 OAuth·쿠팡 HMAC→ 마켓 반영 · 상품·재고 갱신
갈래 A · 신규 상품 등록(모음전 상품을 마켓에 처음 올림)  ·  갈래 B · 기존 상품 연동(이미 올라간 상품과 매칭 → 가격·재고만 동기화, 현재 우선 경로)
