크롤러 설치

본인 PC 크롬에 한 번만 설치하면 무신사·롯데온까지 긁어 모음전에 자동 저장됩니다.
💡 무신사·롯데온은 로그인·SPA라 서버가 직접 못 긁어, 내 로그인 크롬 확장이 대신 긁습니다. 크롤 결과는 모두 www.mou-m.com에 저장돼 팀 전체가 함께 봅니다.
1설치 한 번만
1
📥 확장 파일 다운로드
「모음전 크롤러」 압축파일(zip)을 받습니다.
{% if ext_available %} 📥 모음전 크롤러 다운로드 (.zip) {% else %}
⚠️ 지금은 다운로드를 준비할 수 없어요. 관리자에게 알려주세요.
{% endif %}
2
🗂️ 압축 풀기
받은 zip 우클릭 → "압축 풀기" → moum-crawler 폴더가 생깁니다.
3
🧩 크롬 확장 페이지 열기
주소창에 chrome://extensions 입력 → 오른쪽 위 「개발자 모드」 켜기.
4
📂 확장 불러오기
「압축해제된 확장 프로그램을 로드합니다」 → 방금 푼 moum-crawler 폴더 선택. 목록에 "모음전 크롤러"가 뜨면 완료.
2로그인 & 크롤 매번
5
🔑 무신사·롯데온 로그인
같은 크롬에서 로그인해 둡니다. (회원가를 읽으려면 필수 — 로그아웃 상태면 비회원가로 긁힙니다.)
6
🚀 크롤 실행
www.mou-m.com 접속(IP 주소 ❌) → 상품에서 「전체 크롤」. 이제 무신사·롯데온 카드도 0% → 채워집니다.
🛟 안 될 때 (펼치기 ▾)
Q. 설치했는데 계속 "확장 필요"가 떠요.
주소가 www.mou-m.com인지 확인하세요. IP 주소(54.x)에서는 작동하지 않습니다.
Q. 무신사 가격이 비싸게 나와요.
무신사에 로그인됐는지 확인하세요. 로그아웃 상태면 비회원가로 읽힙니다.
Q. 잘 깔렸는지 확인하고 싶어요.
www.mou-m.com 새로고침 → F12 → Console에 document.documentElement.getAttribute('data-moum-ext') 입력 → 버전 번호가 나오면 성공.
