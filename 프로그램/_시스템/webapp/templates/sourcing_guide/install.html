✓
🗂️ 압축 풀기
받은 zip을 우클릭 → "압축 풀기". moum-crawler 폴더가 생깁니다.
✓
🧩 크롬 확장 페이지 열기
크롬 주소창에 chrome://extensions 입력 → 오른쪽 위 「개발자 모드」 켜기.
✓
📂 확장 불러오기
「압축해제된 확장 프로그램을 로드합니다」 클릭 → 방금 푼 moum-crawler 폴더 선택. 목록에 "모음전 크롤러"가 뜨면 설치 완료.
✓
🔑 무신사·롯데온 로그인
같은 크롬에서 무신사·롯데온에 로그인해 둡니다. (회원가를 읽으려면 필수 — 로그아웃 상태면 비회원가로 긁힙니다.)
✓
🚀 크롤 실행
www.mou-m.com 접속(IP 주소 ❌) → 상품에서 「전체 크롤」. 이제 무신사·롯데온 카드도 0% → 채워집니다.
✅ 설치 후 「전체 크롤」을 누르면 무신사·롯데온이 내 PC에서 긁혀 www.mou-m.com에 저장됩니다.

🛟 안 될 때

Q. 설치했는데 계속 "확장 필요"가 떠요.
주소가 www.mou-m.com인지 확인하세요. IP 주소(54.x)에서는 확장이 작동하지 않습니다.
Q. 무신사 가격이 비싸게 나와요.
무신사에 로그인됐는지 확인하세요. 로그아웃 상태면 비회원가로 읽힙니다.
Q. 잘 깔렸는지 확인하고 싶어요.
www.mou-m.com 새로고침 → F12 → Console에 document.documentElement.getAttribute('data-moum-ext') 입력 → 버전 번호가 나오면 성공.
