### 여백 표준 추정값 (Formtec 공식 미공개 항목)
- **top margin (mt)** = (297 − rows × H) / 2 (위·아래 균등 분배 가정)
- **left margin (ml)** = (210 − cols × W) / 2 (좌·우 균등 분배 가정)
- **gap_x, gap_y** = 0 (인접 라벨 간격 0 가정, 실제 일부 모델은 1~3mm 있음)

> ⚠️ **사용자 알림**: 위 mt/ml/gap 은 공식 데이터 미공개로 인한 추정. 인쇄 후 어긋나면 **Custom 모드** 로 직접 측정·보정 필요.

## ⚙️ Custom 모드 (수기 입력) — 최강 신뢰도

진실 원천이 없거나 보유한 용지가 위 표에 없는 경우 사용자가 직접 측정.

### 측정 방법
1. 자로 라벨 한 칸 가로(W) · 세로(H) 측정
2. 종이 상단부터 첫 라벨 윗변까지 = 상단 여백 (mt)
3. 종이 왼쪽부터 첫 라벨 왼변까지 = 좌측 여백 (ml)
4. 인접 라벨 가로 간격 (gap_x), 세로 간격 (gap_y) 측정 (0 일 수도 있음)
5. 라벨 가로 칸 개수 (cols), 세로 칸 개수 (rows)

### 검산 공식 (A4 = 210 × 297 mm)
- `2 × ml + cols × W + (cols - 1) × gap_x ≈ 210` (가로 검산)
- `2 × mt + rows × H + (rows - 1) × gap_y ≈ 297` (세로 검산)

→ 양변 차이가 1mm 이내면 입력 정확.

## 📞 사용자 피드백 채널

3. 어긋난 정도 (mm)

위 3가지를 알려주시면 PRESETS 또는 본 문서를 즉시 보정합니다.
