# 모음전 크롤러 (크롬 확장)

mou-m.com 에서 '전체 크롤'을 누르면, **로그인된 이 브라우저**로 소싱처 가격을 수집해
서버에 전달하는 크롬 확장입니다. (서버가 직접 크롤하지 않음 — 무신사 로그인·롯데온이
브라우저 세션으로 깔끔히 해결됨. 참고: 더망고솔루션도 동일한 크롬 확장 방식.)

## 설치 (개발용 — 한 번만)

1. 크롬 주소창에 `chrome://extensions` 입력 → 이동
2. 오른쪽 위 **개발자 모드** 토글 **켜기**
3. 왼쪽 위 **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 폴더(`extension/moum-crawler`) 선택
5. 목록에 **"모음전 크롤러"** 가 뜨면 완료

## 동작 확인

mou-m.com 새로고침 후, 페이지 콘솔에서:

```js
document.documentElement.getAttribute('data-moum-ext')   // "0.1.0" 이면 설치 감지 OK
```

## 구조

| 파일 | 역할 |
|---|---|
| `manifest.json` | 확장 설정(권한·소싱처 도메인·콘텐츠 스크립트) |
| `content_mou.js` | mou-m.com 에 설치 마커 심기 + 페이지↔확장 메시지 브리지 |
| `background.js` | 실제 크롤 담당(서비스 워커). Step 1 은 ping, Step 2 에 크롤러 추가 |

## 단계

- **Step 1** (현재): 뼈대 + 설치감지 + 통신 브리지(ping)
- **Step 2**: 무신사·롯데온 크롤러(JS) — 지금 실패하는 2개 먼저
- **Step 3**: 르무통·스마트스토어·SSF·SSG 크롤러
- **Step 4**: 이 PC에서 전체크롤 → 무신사·롯데온 가격+fx 시연
