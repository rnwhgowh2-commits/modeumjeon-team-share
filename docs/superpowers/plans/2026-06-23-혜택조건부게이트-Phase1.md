### Task 1b-4: 라이브 검증 + 동적 크롤·조건부 시연 (필수)
- [ ] **시연①(크롤값)**: 무신사 한 혜택 "크롤값" → 서로 다른 두 상품 영수증서 그 값이 상품마다 다른 크롤값으로 차감 확인. 고정값 혜택=입력값 그대로.
- [ ] **시연②(조건부)**: 무신사 크롤값 혜택에 조건부(적용 "후기"/제외 "불가") → 키워드대로 적용/미적용 → 최종매입가 달라짐 영수증 실대조.
- [ ] **불변**: always(고정값) 혜택 영향0·폴백가 없음·이중차감 없음.
- [ ] 7섹션 보고 + origin/main 머지·배포. (라이브=무신사 실상품, dev DB 불신)
