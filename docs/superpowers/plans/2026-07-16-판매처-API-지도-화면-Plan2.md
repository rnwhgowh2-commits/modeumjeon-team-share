# 판매처 API 지도 — 화면 (Plan 2) 구현 플랜

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development 또는 executing-plans. 단, 이 플랜은 **map.html 단일 파일의 긴밀한 프론트 재작성**이라 태스크가 순차·결합적이다. 병렬 서브에이전트 금지, 순차 실행 + 매 태스크 브라우저 렌더 실측.

**Goal:** 「데이터 코드 지도」 화면(map.html)을 **데이터 파일 구동**으로 바꾸고, 6 업무탭 중 신규 4개(상품 조회·상품 관리·배송·송장·클레임·CS)에 **기능×마켓 비교표 + 줄 아래 상세 펼침**을 추가한다.

**확정 디자인(결정 게이트 통과):** ① 비교 칸 = **API 이름 칩**(상태 색 유지) · ② 상세 = **그 줄 바로 아래 펼침(아코디언)** · ③ 최상위 탭 = **한 줄 알약**.

**Architecture:** map.html이 로드 시 `/marketplace-guide/map-data.json`(Plan 1 산출) 을 fetch → 인메모리 구조로 가공. 「마켓별 전체 API」는 apis 를 market→category 로 묶어 기존 보내기/받기 2열 렌더에 그대로 흘려보냄. 신규 업무탭은 apis 를 `tabs[]`·`fnKey` 로 묶어 비교표+아코디언 렌더. 기존 howto/정산/주문상태 특수 뷰는 보존(기능 손실 0).

**Tech Stack:** 순수 JS(바닐라, 프레임워크 X) · Flask Jinja 템플릿(`{% raw %}` 블록) · pytest(템플릿 존재 검증).

**Spec:** `docs/superpowers/specs/2026-07-16-판매처-마켓별-API-단일진실원천-design.md` §4
**대상 파일(단 하나):** `프로그램/_시스템/webapp/templates/marketplace_guide/map.html` (938줄, `<script>{% raw %}` 내부)

**⚠️ 무결성:** 기존 탭(howto·정산·주문상태)·별표·env 바 동작을 **깨지 않는다**. fetch 실패 시 조용히 빈 화면 금지 → 에러 배너.

## 데이터 형태 (Plan 1 산출 `map-data.json`)

```
apis[]  { id, market, fnKey, tabs[], category, nm, dir, st, endpoint,
          req{}, res{}, fields[{key,meaning,example}], success, idTraps[], persistIds[], codeRef }
markets[] { id, label, base_url, auth, env_prefixes[], concurrency }
```
- `st` ∈ {ok, code, off, todo}. 표시 라벨 = 기존 `DM_LBL`.
- 업무탭 6종 이름(정확): `"상품 조회" "상품 관리" "정산" "배송·송장" "클레임·CS" "주문상태"`.

```js
// ── Plan 2: 단일 진실 원천 데이터 로드 ──
let DATA=null;               // {markets, apis, unifiedStatuses, transitions, codes}
let MK_META={};              // id → market meta
let MKT_CATS_D={};           // id → [{g, items:[api...]}]  (마켓별 전체 API용)
const WORK_TABS=["상품 조회","상품 관리","정산","배송·송장","클레임·CS","주문상태"];

function buildAdapters(){
  MK_META={}; DATA.markets.forEach(m=>MK_META[m.id]=m);
  MKT_CATS_D={};
  DATA.markets.forEach(m=>MKT_CATS_D[m.id]=[]);
  const byMkCat={};
  DATA.apis.forEach(a=>{
    (byMkCat[a.market]=byMkCat[a.market]||{});
    (byMkCat[a.market][a.category]=byMkCat[a.market][a.category]||[]).push(a);
  });
  for(const mk in byMkCat){
    MKT_CATS_D[mk]=Object.keys(byMkCat[mk]).map(g=>({g, items:byMkCat[mk][g]}));
  }
}
async function loadData(){
  try{
    const r=await fetch('/marketplace-guide/map-data.json',{credentials:'same-origin'});
    if(!r.ok) throw new Error('HTTP '+r.status);
    DATA=await r.json();
    if(DATA.validation_errors && DATA.validation_errors.length)
      console.warn('map-data 검증 오류', DATA.validation_errors);
    buildAdapters();
  }catch(e){
    document.getElementById('dm-body').innerHTML=
      `<div class="empty"><div class="big">⚠️</div>데이터를 불러오지 못했어요.<br><span style="font-size:11px;color:var(--faint)">${e.message}</span></div>`;
    throw e;
  }
}
```

- [ ] **Step 2: 렌더 함수 추가** — `dmRender` 정의 위에:
```js
function dmMkList(){return DATA.markets.map(m=>m.id);}
function esc(v){return (typeof v==='string'?v:JSON.stringify(v,null,2)).replace(/&/g,'&amp;').replace(/</g,'&lt;');}
function dmWorkTab(tab){
  const list=DATA.apis.filter(a=>(a.tabs||[]).includes(tab));
  // fnKey 로 묶기(대표 nm/dir = 첫 항목)
  const byFn={}; list.forEach(a=>{(byFn[a.fnKey]=byFn[a.fnKey]||[]).push(a);});
  const mks=dmMkList();
  let h=`<div class="favhead"><span class="ft">${tab}</span><span class="fs">기능별로 마켓이 주는지(칩=실제 API 이름·색=상태) / 칸을 누르면 아래에 실제 요청·응답이 펼쳐집니다</span></div>`;
  h+=`<table class="wtbl"><thead><tr><th class="fn">기능</th>`+mks.map(id=>`<th class="mk">${MK_META[id].label}</th>`).join('')+`</tr></thead><tbody>`;
  Object.keys(byFn).forEach(fn=>{
    const rep=byFn[fn][0];
    let cells='';
    mks.forEach(id=>{
      const a=byFn[fn].find(x=>x.market===id);
      if(!a){ cells+=`<td class="mk"><span class="wchip none">—</span></td>`; }
      else{ const short=(a.endpoint||a.id).split(' ').pop().split('/').slice(-2).join('/');
        cells+=`<td class="mk"><span class="wchip ${a.st}" data-id="${a.id}">${short}</span></td>`; }
    });
    h+=`<tr class="wrow" data-fn="${fn}"><td class="fn"><span class="dir">${rep.dir==='recv'?'받기':'보내기'}</span>${rep.nm}</td>${cells}</tr>`;
  });
  h+=`</tbody></table>`;
  return h;
}
function dmApiDetail(a){
  if(!(a.st==='ok'||a.st==='code')||(!a.req&&!a.res)){
    return `<div class="wd"><div class="todo">아직 실제 요청/응답이 안 채워졌어요. (상태: ${DM_LBL[a.st]||a.st}) — 이 마켓 API 문서를 주시면 채웁니다.</div></div>`;
  }
  let f='';
  (a.fields||[]).forEach(x=>f+=`<tr><td class="key">${x.key}</td><td>${x.meaning}</td><td class="key">${esc(x.example)}</td></tr>`);
  return `<div class="wd">
    <div class="k">실제 요청</div><pre>${a.endpoint||''}\n${esc(a.req)}</pre>
    <div class="k">실제 응답</div><pre>${esc(a.res)}</pre>
    ${f?`<div class="k">응답 필드 뜻</div><table class="f"><thead><tr><th>키</th><th>뜻</th><th>예시</th></tr></thead><tbody>${f}</tbody></table>`:''}
    <div class="succ"><b>성공 판정</b> — ${a.success||'—'}</div>
  </div>`;
}
function bindWorkRows(){
  document.querySelectorAll('.dm2 .wchip[data-id]').forEach(ch=>ch.onclick=(e)=>{
    e.stopPropagation();
    const a=DATA.apis.find(x=>x.id===ch.dataset.id);
    const tr=ch.closest('tr');
    const nxt=tr.nextElementSibling;
    if(nxt&&nxt.classList.contains('wdetail')){ nxt.remove(); if(nxt.dataset.id===a.id) return; }
    document.querySelectorAll('.dm2 .wdetail').forEach(x=>x.remove());
    const colspan=DATA.markets.length+1;
    const drow=document.createElement('tr'); drow.className='wdetail'; drow.dataset.id=a.id;
    drow.innerHTML=`<td colspan="${colspan}">${dmApiDetail(a)}</td>`;
    tr.after(drow);
  });
}
```
