# 적용 범위 5항목 통일 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 모음전 상세 계산식 팝업의 혜택 추가·삭제·수정 3곳을 동일한 5항목 적용범위(option/select/bundle/bundle_all_src/source)로 통일하고, "옵션 매트릭스 직접 선택" 팝업을 추가한다.

**Architecture:** 프런트는 공통 헬퍼 `renderScopePicker`(세로 라디오 카드, 시안 2)로 3곳을 묶고, "옵션 매트릭스 직접 선택"은 전역 `DATA.options`로 색×사이즈 그리드를 그려 선택 sku 목록을 반환한다. 백엔드는 새 테이블 없이 `OptionBenefitOverride`/`SourceBenefitTemplate`에 N건 insert/delete하는 scope 분기 2종(`select`, `bundle_all_src`)을 추가한다.

**Tech Stack:** Flask(Python) + SQLAlchemy + Supabase Postgres(dev) / SQLite(live), Jinja + vanilla JS(`_matrix_v3.html`), pytest + Flask test client.

**작업 디렉터리:** `프로그램/_시스템` (이하 모든 경로 기준). 테스트 실행도 이 폴더에서.

```python
def _upsert_value_for_skus(s, source_id, skus, nm, bt, val):
    tpl = (s.query(SourceBenefitTemplate)
           .filter_by(source_id=source_id, benefit_name=nm).first())
    tpl_id = tpl.id if tpl else None
    affected = 0
    for sku in skus:
        existing = (s.query(OptionBenefitOverride)
                    .filter_by(source_id=source_id, canonical_sku=sku, benefit_name=nm).first())
        if existing:
            existing.benefit_type = bt; existing.value = val; existing.enabled = 1
            if tpl_id and not existing.template_id:
                existing.template_id = tpl_id
            if tpl is not None:
                existing.category = tpl.category; existing.apply_mode = tpl.apply_mode
                existing.pay_method = tpl.pay_method; existing.channel = tpl.channel
        else:
            last = (s.query(OptionBenefitOverride)
                    .filter_by(canonical_sku=sku, source_id=source_id)
                    .order_by(OptionBenefitOverride.sort_order.desc()).first())
            s.add(OptionBenefitOverride(
                canonical_sku=sku, source_id=source_id, template_id=tpl_id,
                benefit_name=nm, benefit_type=bt, value=val,
                category=(tpl.category if tpl else None),
                apply_mode=(tpl.apply_mode if tpl else None),
                pay_method=(tpl.pay_method if tpl else None),
                channel=(tpl.channel if tpl else None),
                enabled=1, sort_order=((last.sort_order + 1) if last else 0)))
        affected += 1
    return affected
```

`<script>` 영역(매트릭스 JS 시작부, 예: `function smOpenScopeModal` 위)에 추가. 5항목 세로 라디오 카드를 문자열로 반환:

```javascript
// 적용범위 5항목 공통 picker (시안 2 · 세로 라디오 카드)
//   ctx = {bundleCount, srcName, curColor, defaultScope}
//   반환 = HTML 문자열. 컨테이너에 넣은 뒤 wireScopePicker(el, onPick) 로 배선.
function renderScopePicker(ctx) {
  const def = ctx.defaultScope || 'bundle';
  const items = [
    {scope:'option',         dot:'g', lbl:'이 옵션만',                cnt:'+1'},
    {scope:'select',         dot:'t', lbl:'옵션 매트릭스 직접 선택',  act:'선택 →'},
    {scope:'bundle',         dot:'b', lbl:'모음전 전체',              cnt:'+'+(ctx.bundleCount||0)},
    {scope:'bundle_all_src', dot:'o', lbl:'해당 모음전 소싱처 전체',  cnt:'다수'},
    {scope:'source',         dot:'p', lbl:'프로그램 소싱처 전체',     cnt:'다수'},
  ];
  return `<div class="sp-list">` + items.map(it => `
    <button type="button" class="sp-row${it.scope===def?' on':''}" data-scope="${it.scope}">
      <span class="sp-rdo"></span><span class="sp-dot ${it.dot}"></span>
      <span class="sp-lbl">${it.lbl}</span>
      ${it.act ? `<span class="sp-act">${it.act}</span>` : `<span class="sp-cnt">${it.cnt}</span>`}
    </button>`).join('') + `</div>`;
}

// 선택 상태 머신. el=컨테이너. opts.onSelect(scope) 콜백.
//   select 선택 시 매트릭스 팝업 오픈 → 선택 sku[] 저장. getState() 로 조회.
function wireScopePicker(el, opts) {
  opts = opts || {};
  let scope = el.querySelector('.sp-row.on')?.dataset.scope || 'bundle';
  let pickedSkus = [];
  el.querySelectorAll('.sp-row').forEach(row => row.addEventListener('click', () => {
    el.querySelectorAll('.sp-row').forEach(r => r.classList.remove('on'));
    row.classList.add('on');
    scope = row.dataset.scope;
    if (scope === 'select') {
      openOptionMatrixPicker(pickedSkus, (skus) => {
        pickedSkus = skus;
        row.querySelector('.sp-act').textContent = skus.length ? `${skus.length}개 선택됨 · 변경` : '선택 →';
        if (opts.onSelect) opts.onSelect(scope, pickedSkus);
```
