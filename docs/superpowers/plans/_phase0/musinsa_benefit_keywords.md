## 관찰 (잠정)

- **표면노출가 = salePrice** 확정(122,900 = 화면). 표면가 출처 결정(API 유지) 검증 OK(상태 A).
- **off 신호 키워드**: `불가`, `없음`, `적용 안함` → exclude_keywords 후보.
- **금액은 라인 텍스트에 내장**(`-8,600원`, `(+4,570원)`, `-4,000원`) → 파서가 라인에서 추출 가능(Phase 2b).
- **계정 의존 혜택**(등급할인·등급적립·회원가·적립금)은 로그인 상태/LV에 따라 변동 예상 → 상태 B/C 대조 필요.
- **결제수단 즉시할인은 택1**(카카오 -4,000 vs 토스 -5,000), 조건부(금액 기준).

## 미결(사용자 도메인 확인 필요)

- `나의 할인가` 110,300 ↔ 표면가 122,900 의 12,600 차이의 정체(어떤 혜택이 회원가에 선반영?).
- 매입가 산정 시 `적립금 사용`(본인 보유분)·`구매적립`(적립)을 차감 혜택으로 볼지 여부.
- 각 혜택의 계산속성 최종 확정(정률/정액/조건부, 베이스순서, 상시/기간, 1회성/무제한).
