## 무변화 갱신 설계 (사장님 지시: "무변화 갱신, 안 되면 재고 1개 올리고 내리고")

### 원칙

1. **가격은 어떤 경우에도 건드리지 않는다.** 재고 필드만. (가격 오류 = 즉시 금전 손실)
2. **현재값을 읽어 그 값 그대로 쓴다.** 전송 전 조회 → 같은 값 PUT → 사후 조회로 값이 안 바뀐 것 확인.
3. **대상은 지정된 테스트 상품 1개뿐.** 상품 ID 화이트리스트 밖은 `ProbeForbidden`.
4. 무변화가 마켓에서 거부되거나(`변경사항 없음` 에러) 429 판정에 영향이 없으면 → **±1 토글**로 전환.
   토글은 **반드시 짝수 회**로 끝내 원래 값으로 복귀시키고, 마지막에 사후 조회로 검증한다.

### 🔴 재고 ±1 토글의 위험 — 반드시 판매중지 상품으로

토글은 순간적으로 재고를 +1 한다. **노출 중인 상품이면 그 사이 주문이 들어와 없는 재고를 판다**(오버셀).
→ 테스트 대상은 **판매중지·미노출 상태 상품**이어야 한다. 사장님이 마켓마다 1개씩 만들어 주셔야 한다.

무변화 갱신만 쓴다면 재고가 바뀌지 않으므로 노출 상품이어도 이론상 안전하지만,
프로브가 중간에 죽으면 마지막 상태를 보장할 수 없으므로 **테스트 상품 사용을 권장**한다.

## 측정 설계 (점진 증속)

엔드포인트마다 3개 값을 구한다.

1. **버스트 용량 C** — 120초 완전 휴식 후 간격 없이 연속 호출. 첫 429 직전까지 성공 수.
2. **지속 속도 R** — **점진 증속(램프업)**: 0.2 → 0.5 → 1 → 2 → 3 → 5 → 8 → 12 → 20 req/s 순으로
   각 단계를 30초 유지. 429 가 처음 나온 **직전 단계**가 안전 상한. 그 뒤 이분탐색으로 정밀화.
   - 계단식으로 올리는 이유: 이분탐색만 하면 첫 시도가 20 req/s 라 곧바로 차단당할 수 있다.
3. **회복 시간 T** — 429 직후 1초 간격 단발 호출, 첫 200 시각. 분 경계에 몰리면 고정창.

최종 표기: **"C개 버스트 후 R req/s 지속"**. `분당 N회` 로 뭉개지 않는다.
**우리 제한장치에 넣을 값 = R × 0.7** (안전마진 30%). 마진은 `config.py` 상수로 조정 가능.

## 제한 단위 판별 (계정별 / IP별 / 계정×IP / 전역)

A@IP1 을 429 까지 소진한 **직후 3초 이내**에:
- **P1** = B@IP1 (다른 계정, 같은 IP) 1회
- **P2** = A@IP2 (같은 계정, 다른 IP) 1회

| P1 | P2 | 결론 | 제한장치 설계 |
|---|---|---|---|
| 200 | 429 | **계정(키)별** | 계정 합산 방식이 맞다. 마켓 캡 불필요 |
| 429 | 200 | **IP별** | **계정 합산은 틀렸다.** 마켓 전체 캡이 필수 |
| 200 | 200 | 계정×IP 쌍별 | 계정 합산 + IP 수만큼 곱 |
| 429 | 429 | 전역 | 마켓 전체 캡 필수, 계정 무의미 |

429 가 아닌 응답(403 = IP 미등록 등)은 절대 429 로 간주하지 않고 `판별불가`.

```python
# -*- coding: utf-8 -*-
"""프로브 안전장치.

★ 가격은 어떤 경우에도 건드리지 않는다. 재고 필드만.
★ 여기 등재된 테스트 상품 외에는 절대 대상으로 삼지 않는다.
"""
from __future__ import annotations

import os
from pathlib import Path

STOP_FILE = Path(__file__).with_name("STOP")
DEFAULT_BUDGET = 2000
CONSECUTIVE_429_ABORT = 20
MARKET_GAP_SEC = 300
BURST_REST_SEC = 120

# 램프업 계단 (req/s). 이분탐색만 하면 첫 시도가 20 이라 곧바로 차단당한다.
RAMP_STEPS = (0.2, 0.5, 1.0, 2.0, 3.0, 5.0, 8.0, 12.0, 20.0)
HOLD_SEC = 30.0          # 각 계단 유지 시간
COOLDOWN_SEC = 60.0      # 계단 사이 휴식


class ProbeForbidden(RuntimeError):
    """화이트리스트 밖 대상·동작."""


# ★ 사장님이 마켓마다 만들어 주신 **판매중지·미노출** 테스트 상품만 여기 넣는다.
#   비어 있으면 그 마켓은 측정하지 않는다(추정값 금지).
TEST_TARGETS: dict[str, dict] = {
    # "coupang":  {"product_id": "", "option_id": ""},   # vendorItemId
    # "lotteon":  {"product_id": "", "option_id": ""},   # spdNo / sitmNo
    # "eleven11": {"product_id": "", "option_id": ""},
    # "smartstore": {"product_id": "", "option_id": ""}, # originProductNo / optionId
    # "auction":  {"product_id": "", "option_id": ""},
    # "gmarket":  {"product_id": "", "option_id": ""},
}

# 「1건 업로드」에 실제로 나가는 API 호출 수.
#   스스·ESM 은 현재값을 GET 한 뒤 전체를 PUT 하므로 2회다.
_CALLS_PER_UPLOAD = {
    "coupang": 1, "lotteon": 1, "eleven11": 1,
    "smartstore": 2, "auction": 2, "gmarket": 2,
}


def assert_target_allowed(market: str, *, product_id: str, option_id: str) -> None:
    t = TEST_TARGETS.get(market)
    if not t:
        raise ProbeForbidden(
            f"{market}: 테스트 상품 미등록 — 측정 불가(추정값 금지)")
    if str(product_id) != str(t["product_id"]) or str(option_id) != str(t["option_id"]):
        raise ProbeForbidden(
            f"{market}: 화이트리스트 밖 대상 product={product_id} option={option_id}")


def calls_per_upload(market: str) -> int:
    if market not in _CALLS_PER_UPLOAD:
        raise ProbeForbidden(f"미등록 마켓: {market}")
    return _CALLS_PER_UPLOAD[market]


def safety_margin() -> float:
    """실측 상한에 곱할 안전계수. 경계값을 그대로 쓰면 429 가 난다."""
    return float(os.getenv("PROBE_SAFETY_MARGIN", "0.7"))


def budget_for(market: str) -> int:
    return int(os.getenv(f"PROBE_BUDGET_{market.upper()}", DEFAULT_BUDGET))


def stop_requested() -> bool:
    return STOP_FILE.exists()
```

```python
# -*- coding: utf-8 -*-
"""프로브 호출 기록기 — 시각·상태·응답헤더 전부를 JSONL 로.

헤더를 통째로 남기는 이유: 어떤 마켓이 어떤 이름으로 잔여량을 알려주는지 모른다.
시크릿이 섞일 수 있는 헤더만 마스킹한다.
"""
from __future__ import annotations

import json
import time
from pathlib import Path

_SECRET_HEADERS = {
    "authorization", "openapikey", "x-api-key", "cookie", "set-cookie",
    "api-key", "apikey", "x-auth-token",
}


class Recorder:
    def __init__(self, out_path: Path, *, market: str, account: str, ip_label: str):
        self._path = Path(out_path)
        self._path.parent.mkdir(parents=True, exist_ok=True)
        self._market, self._account, self._ip_label = market, account, ip_label

    def record(self, method: str, url: str, resp, *, elapsed_ms: float,
               phase: str = "") -> None:
        row = {
            "ts": time.time(), "market": self._market, "account": self._account,
            "ip_label": self._ip_label, "phase": phase,
            "method": method, "url": url,
            "status": getattr(resp, "status_code", None),
            "elapsed_ms": round(float(elapsed_ms), 2),
            "headers": self._clean(getattr(resp, "headers", {}) or {}),
            "body_head": (getattr(resp, "text", "") or "")[:200],
        }
        with self._path.open("a", encoding="utf-8") as f:
            f.write(json.dumps(row, ensure_ascii=False) + "\n")

    @staticmethod
    def _clean(headers) -> dict:
        return {k: ("***REDACTED***" if k.lower() in _SECRET_HEADERS else v)
                for k, v in dict(headers).items()}
```

```python
# -*- coding: utf-8 -*-
"""무변화 갱신 — 재고를 현재값 그대로 다시 쓴다.

★ 가격은 절대 건드리지 않는다. 재고 필드만.
★ 현재값을 못 읽으면 쓰지 않는다 — 무변화를 보장할 수 없으므로.
"""
from __future__ import annotations

import time
from dataclasses import dataclass
from typing import Optional

from .config import TEST_TARGETS, assert_target_allowed, ProbeForbidden
from .recorder import Recorder


@dataclass(frozen=True)
class WriteResult:
    status: Optional[int]
    is_rate_limited: bool
    retry_after: Optional[int]
    elapsed_ms: float
    error: Optional[str] = None


def _client_for(market: str, account: str):
    """리미터·재시도를 끈 클라이언트. 429 를 삼키면 측정이 불가능하다."""
    from lemouton.auth import secrets as S
    from shared import platforms as P

    ov = {"max_retries": 1, "retry_backoff_sec": 0, "rate_limit_per_sec": 10 ** 6}
    if market in ("auction", "gmarket"):
        ov["order_min_interval_sec"] = 0

    if market == "coupang":
        from shared.platforms.coupang.client import CoupangClient
        c = S.load_credentials(market="coupang", env_prefix=account)
        return CoupangClient(config={**P.COUPANG, "access_key": c.access_key,
                                     "secret_key": c.secret_key,
                                     "vendor_id": c.vendor_id, **ov})
    if market == "lotteon":
        from shared.platforms.lotteon.client import LotteonClient
        c = S.load_credentials(market="lotteon", env_prefix=account)
        return LotteonClient(config={**P.LOTTEON, "api_key": c.api_key,
                                     "tr_no": c.tr_no, **ov})
    if market == "eleven11":
        from shared.platforms.eleven11.client import Eleven11Client
        c = S.load_credentials(market="eleven11", env_prefix=account)
        return Eleven11Client(config={**P.ELEVEN11, "openapi_key": c.openapi_key, **ov})
    if market == "smartstore":
        from shared.platforms.smartstore.client import SmartStoreClient
        c = S.load_credentials(market="smartstore", env_prefix=account)
        return SmartStoreClient(config={**P.SMARTSTORE, "client_id": c.client_id,
                                        "client_secret": c.client_secret, **ov})
    if market in ("auction", "gmarket"):
        from shared.platforms.esm.client import EsmClient
        base = P.AUCTION if market == "auction" else P.GMARKET
        c = S.load_credentials(market=market, env_prefix=account)
        return EsmClient(config={**base, "master_id": c.master_id,
                                 "secret_key": c.secret_key,
                                 "seller_id": c.seller_id, **ov})
    raise ProbeForbidden(f"미지원 마켓: {market}")


def _read_stock(market: str, *, client, product_id: str, option_id: str):
    """현재 재고. 못 읽으면 None (테스트에서 monkeypatch 대상)."""
    if market == "coupang":
        from shared.platforms.coupang.inventory import get_quantity
        return get_quantity(int(option_id), client=client)
    from lemouton.uploader.market_fetch import fetch_market_options
    r = fetch_market_options(market, str(product_id))
    if not r.success:
        return None
    for o in r.options:
        if str(o.option_id) == str(option_id):
            return o.stock
    return None


def _write_stock(market: str, *, client, product_id: str, option_id: str, stock: int):
    """재고만 기록. 가격 인자는 어디에도 넘기지 않는다."""
    if market == "coupang":
        from shared.platforms.coupang.inventory import update_quantity
        update_quantity(int(option_id), int(stock), client=client)
    elif market == "lotteon":
        from shared.platforms.lotteon.inventory import update_stock
        update_stock(str(product_id), str(option_id), int(stock), client=client)
    elif market == "eleven11":
        from shared.platforms.eleven11.inventory import update_stock
        update_stock(str(option_id), int(stock), client=client)
    elif market in ("auction", "gmarket"):
        from shared.platforms.esm.inventory import update_stock
        update_stock(str(product_id), market, str(option_id), int(stock), client=client)
    elif market == "smartstore":
        from shared.platforms.smartstore.edit_product import edit_options
        edit_options(int(product_id),
                     option_updates={int(option_id): {"stockQuantity": int(stock)}},
                     client=client)          # sale_price 는 넘기지 않는다(현재값 유지)
    else:
        raise ProbeForbidden(f"미지원 마켓: {market}")
    return _LastResponse.get(client)


class _LastResponse:
    """클라이언트가 마지막으로 받은 응답 — 상태·헤더를 얻기 위한 훅."""
    @staticmethod
    def get(client):
        return getattr(client, "_last_response", None)


def noop_write(market: str, *, account: str, recorder: Recorder,
               client=None, ip_label: str = "IP1") -> WriteResult:
    """현재값을 읽어 **그 값 그대로** 다시 쓴다. 1회."""
    t = TEST_TARGETS.get(market)
    if not t:
        raise ProbeForbidden(f"{market}: 테스트 상품 미등록")
    assert_target_allowed(market, product_id=t["product_id"], option_id=t["option_id"])

    client = client or _client_for(market, account)
    cur = _read_stock(market, client=client,
                      product_id=t["product_id"], option_id=t["option_id"])
    if cur is None:
        return WriteResult(None, False, None, 0.0,
                           error="현재값을 못 읽어 무변화를 보장할 수 없음 — 쓰지 않음")

    t0 = time.monotonic()
    try:
        resp = _write_stock(market, client=client, product_id=t["product_id"],
                            option_id=t["option_id"], stock=int(cur))
    except Exception as e:  # noqa: BLE001
        return WriteResult(None, False, None, (time.monotonic() - t0) * 1000, str(e))
    elapsed = (time.monotonic() - t0) * 1000

    status = getattr(resp, "status_code", 200) if resp is not None else 200
    if resp is not None:
        recorder.record("PUT", getattr(resp, "url", market), resp,
                        elapsed_ms=elapsed, phase="noop")
    ra = None
    if status == 429 and resp is not None:
        try:
            v = resp.headers.get("Retry-After")
            ra = int(v) if v else None
        except (ValueError, AttributeError):
            ra = None
    return WriteResult(status, status == 429, ra, elapsed)


class Toggler:
    """무변화가 안 통할 때의 대안 — 재고 ±1 토글.

    ★ 반드시 짝수 회로 끝내야 원래 값으로 돌아온다. ``at_original`` 로 확인한다.
    ★ 대상은 판매중지·미노출 상품이어야 한다(노출 상품이면 오버셀 위험).
    """

    def __init__(self, market: str, *, account: str, recorder: Recorder,
                 base_stock: int, client=None, ip_label: str = "IP1"):
        self.market = market
        self.account = account
        self.recorder = recorder
        self.base = int(base_stock)
        self.client = client or _client_for(market, account)
        self.ip_label = ip_label
        self._n = 0

    @property
    def at_original(self) -> bool:
        return self._n % 2 == 0

    def step(self) -> WriteResult:
        t = TEST_TARGETS[self.market]
        assert_target_allowed(self.market, product_id=t["product_id"],
                              option_id=t["option_id"])
        target = self.base + 1 if self._n % 2 == 0 else self.base
        t0 = time.monotonic()
        try:
            resp = _write_stock(self.market, client=self.client,
                                product_id=t["product_id"],
                                option_id=t["option_id"], stock=target)
        except Exception as e:  # noqa: BLE001
            return WriteResult(None, False, None, (time.monotonic() - t0) * 1000, str(e))
        self._n += 1
        elapsed = (time.monotonic() - t0) * 1000
        status = getattr(resp, "status_code", 200) if resp is not None else 200
        if resp is not None:
            self.recorder.record("PUT", getattr(resp, "url", self.market), resp,
                                 elapsed_ms=elapsed, phase="toggle")
        return WriteResult(status, status == 429, None, elapsed)
```

```python
class ProbeAborted(RuntimeError):
    """안전장치 발동으로 중단."""


class ProbeGuard:
    """업로드 1건마다 tick() → 응답마다 observe(status).

    예산은 **API 호출 수** 기준이라 마켓별 호출배수(스스·ESM=2)를 곱해 깎는다.
    """

    def __init__(self, *, market: str, budget: int = None,
                 max_consecutive_429: int = None):
        from .config import budget_for, calls_per_upload, CONSECUTIVE_429_ABORT
        self.market = market
        self.budget = int(budget if budget is not None else budget_for(market))
        self.per_upload = calls_per_upload(market)
        self.max_consecutive_429 = int(max_consecutive_429 or CONSECUTIVE_429_ABORT)
        self.used = 0
        self.consecutive_429 = 0

    def tick(self) -> None:
        from . import config
        if config.STOP_FILE.exists():
            raise ProbeAborted(f"[{self.market}] 킬스위치 파일 감지 — 중단")
        if self.consecutive_429 >= self.max_consecutive_429:
            raise ProbeAborted(f"[{self.market}] 429 연속 {self.consecutive_429}회 — 중단")
        if self.used + self.per_upload > self.budget:
            raise ProbeAborted(f"[{self.market}] 호출 예산 {self.budget} 소진 — 중단")
        self.used += self.per_upload

    def observe(self, status) -> None:
        self.consecutive_429 = self.consecutive_429 + 1 if status == 429 else 0
```

```python
# scripts/ratelimit_probe/measure.py
# -*- coding: utf-8 -*-
"""램프업 측정 — 버스트 용량 C, 지속 속도 R, 회복 시간 T.

계단식으로 올린다. 이분탐색만 하면 첫 시도가 20 req/s 라 곧바로 차단당한다.
"""
from __future__ import annotations

import time
from dataclasses import dataclass
from typing import Callable, Optional

from .config import RAMP_STEPS, HOLD_SEC, COOLDOWN_SEC, BURST_REST_SEC, safety_margin


@dataclass
class BurstResult:
    capacity: int
    hit_429: bool
    note: str = ""


@dataclass
class RampResult:
    last_ok: Optional[float]
    first_fail: Optional[float]


def measure_burst(probe: Callable[[], "WriteResult"], *,      # noqa: F821
                  max_calls: int = 200,
                  rest_sec: float = BURST_REST_SEC) -> BurstResult:
    """완전 휴식 후 간격 없이 연속 → 첫 429 직전까지의 성공 수."""
    if rest_sec:
        time.sleep(rest_sec)
    ok = 0
    for _ in range(max_calls):
        r = probe()
        if r.is_rate_limited:
            return BurstResult(ok, True)
        if r.status is None or r.status >= 400:
            return BurstResult(ok, False,
                               f"429 아닌 오류 status={r.status} — 중단(판별불가)")
        ok += 1
    return BurstResult(ok, False, "상한 미도달 — max_calls 를 올려 재측정 필요")


def ramp_up(holds_at: Callable[[float], bool],
            steps: tuple = RAMP_STEPS) -> RampResult:
    """계단식 증속. 처음 실패한 단계와 그 직전 성공 단계를 돌려준다."""
    last_ok = None
    for rate in steps:
        if holds_at(rate):
            last_ok = rate
        else:
            return RampResult(last_ok, rate)
    return RampResult(last_ok, None)


def refine(holds_at: Callable[[float], bool], *, lo: float, hi: float,
           rounds: int = 8) -> float:
    """last_ok ~ first_fail 사이를 이분탐색으로 좁힌다."""
    for _ in range(rounds):
        mid = (lo + hi) / 2
        if holds_at(mid):
            lo = mid
        else:
            hi = mid
    return round(lo, 2)


def make_hold_check(probe: Callable[[], "WriteResult"], *,     # noqa: F821
                    hold_sec: float = HOLD_SEC,
                    cooldown_sec: float = COOLDOWN_SEC) -> Callable[[float], bool]:
    """rate req/s 로 hold_sec 유지해 429 가 0건이면 True."""
    def holds_at(rate: float) -> bool:
        interval = 1.0 / rate
        deadline = time.monotonic() + hold_sec
        while time.monotonic() < deadline:
            if probe().is_rate_limited:
                time.sleep(cooldown_sec)
                return False
            time.sleep(interval)
        time.sleep(cooldown_sec)
        return True
    return holds_at


def recommended_rate(measured_max: float) -> float:
    """제한장치에 넣을 값 = 실측 상한 × 안전마진. 경계값을 그대로 쓰면 429 가 난다."""
    return round(float(measured_max) * safety_margin(), 2)


def classify_recovery(*, t_429: float, t_first_200: float,
                      tolerance_sec: float = 2.0) -> str:
    sec_in_min = t_first_200 % 60.0
    near = sec_in_min <= tolerance_sec or sec_in_min >= (60.0 - tolerance_sec)
    if near and (t_first_200 - t_429) > tolerance_sec:
        return "고정창(분 경계)"
    return "토큰버킷/슬라이딩"


def measure_recovery(probe: Callable[[], "WriteResult"], *,    # noqa: F821
                     t_429: float, max_wait_sec: float = 120.0) -> Optional[float]:
    deadline = time.monotonic() + max_wait_sec
    while time.monotonic() < deadline:
        if probe().status == 200:
            return time.time()
        time.sleep(1.0)
    return None
```

```python
# -*- coding: utf-8 -*-
"""제한 단위 판별 — 계정별 / IP별 / 계정×IP / 전역.

A@IP1 을 429 까지 소진한 직후 3초 이내에
  P1 = B@IP1 (다른 계정, 같은 IP)
  P2 = A@IP2 (같은 계정, 다른 IP)
를 각 1회. 429 가 아닌 응답은 절대 429 로 간주하지 않는다.
"""
from __future__ import annotations

from dataclasses import dataclass
from typing import Optional


@dataclass(frozen=True)
class LimitKeyVerdict:
    verdict: str
    reason: str


_TABLE = {
    (False, True): ("계정(키)별", "다른 계정은 통과, 같은 계정은 다른 IP에서도 막힘"),
    (True, False): ("IP별", "다른 계정도 막힘, 같은 계정이 다른 IP에서는 통과"),
    (False, False): ("계정×IP 쌍별", "다른 계정도 다른 IP도 모두 통과"),
    (True, True): ("판매자/마켓 전역", "다른 계정도 다른 IP도 모두 막힘"),
}

# 제한이 IP/전역 단위면 계정 합산으로 늘리는 게 위험 → 마켓 전체 캡이 필수다.
_CAP_REQUIRED = {"IP별", "판매자/마켓 전역"}


def classify_limit_key(*, p1_status: Optional[int],
                       p2_status: Optional[int]) -> LimitKeyVerdict:
    if p1_status is None:
        return LimitKeyVerdict("판별불가", "P1 미실행 — 두 번째 계정이 없음")
    if p2_status is None:
        return LimitKeyVerdict("판별불가", "P2 미실행 — 두 번째 IP 미확보")
    for label, st in (("P1", p1_status), ("P2", p2_status)):
        if st not in (200, 429):
            return LimitKeyVerdict(
                "판별불가",
                f"{label} 응답이 {st} — 429/200 이 아니라 판정 불가"
                f"{' (IP 미등록 의심)' if st == 403 else ''}")
    verdict, reason = _TABLE[(p1_status == 429, p2_status == 429)]
    return LimitKeyVerdict(verdict, reason)


def market_cap_required(verdict: str) -> bool:
    """이 판정이면 마켓 전체 캡이 반드시 필요한가."""
    return verdict in _CAP_REQUIRED
```
