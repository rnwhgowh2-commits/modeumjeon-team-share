# 무신사 상품쿠폰 · 쿠폰별 키워드 게이트 구현 플랜

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 무신사 상품쿠폰을 "쿠폰별 목록"으로 전량 크롤하고, 서버 로직이 제외 키워드에 안 걸린 쿠폰 중 최고를 골라 최종매입가(fx)에 반영한다(없으면 0원).

**Architecture:** 크롤(확장)=판단 없이 쿠폰 전량 원본 수집 → 저장(`product_coupon_list`) → 계산(compute_breakdown이 상품쿠폰 혜택의 포함/제외 키워드로 쿠폰명 필터 + 최고 선택) → 화면(③ 혜택 편집 10번 재구성 + 영수증 투명성). 키워드는 계산 시점 적용이라 재크롤 불필요.

**Tech Stack:** Python(Flask), pytest, 크롬 확장(JS), Jinja 템플릿 + 바닐라 JS. 기준 브랜치 = `origin/main`.

**스펙:** [docs/superpowers/specs/2026-07-04-무신사-상품쿠폰-쿠폰별-키워드게이트-design.md](../specs/2026-07-04-무신사-상품쿠폰-쿠폰별-키워드게이트-design.md)

def pick_best_coupon(coupons, benefit, exclude_rules=None):
    """상품쿠폰 목록에서 포함/제외 키워드에 살아남은 '최고 금액' 쿠폰 1개.

    coupons: [{'name': str, 'amount': int|float}, ...]  (크롤 원본 전량)
    benefit: 가이드의 상품쿠폰 혜택 dict (triggers / match / excludes / exclude_match)
    exclude_rules: 소싱처 공통 제외 규칙(crawl_guide['exclude_keywords']), 선택.
    반환: {'name','amount','candidates':[...],'excluded':[{'name','amount','reason'}]}
          또는 후보 없음 시 None.
    """
    triggers = benefit.get('triggers') or []
    match = benefit.get('match') or 'any'
    b_excludes = benefit.get('excludes') or []
    b_exmatch = benefit.get('exclude_match') or 'any'
    exclude_rules = exclude_rules or []

    survivors, excluded = [], []
    for c in (coupons or []):
        name = (c.get('name') or '')
        amount = float(c.get('amount') or 0)
        if amount <= 0:
            continue
        if not line_matches_triggers(name, triggers, match):
            excluded.append({'name': name, 'amount': amount, 'reason': '포함 키워드 미매칭'})
            continue
        hit = line_excluded(name, exclude_rules)
        if hit is not None:
            excluded.append({'name': name, 'amount': amount,
                             'reason': f"공통제외 '{hit.get('word')}'"})
            continue
        if line_excluded_by_benefit(name, b_excludes, b_exmatch):
            excluded.append({'name': name, 'amount': amount, 'reason': '제외 키워드'})
            continue
        survivors.append({'name': name, 'amount': amount})

    if not survivors:
        return None
    best = max(survivors, key=lambda x: x['amount'])
    return {'name': best['name'], 'amount': best['amount'],
            'candidates': survivors, 'excluded': excluded}
```

        if (!t || t.length > 80 || !/쿠폰/.test(t)) return;
        const m = t.match(/([\-−]?\s*[\d,]{3,})\s*원/);
        if (!m) return;
        const amount = parseInt(m[1].replace(/[^\d]/g, ''), 10) || 0;
        if (amount <= 0) return;
        const name = t.replace(/[\-−]?\s*[\d,]{3,}\s*원.*/, '').trim().slice(0, 60);
        const key = name + '|' + amount;
        if (name && !seen.has(key)) { seen.add(key); out.push({ name, amount }); }
      });
      return out;
    } catch (e) { return []; }
  }
  const _productCoupons = collectProductCoupons();
```

}
// keydown: Enter 또는 붙여넣기(쉼표 포함) 시 addChipsFromInput 호출
input.addEventListener('keydown', e => {

input.addEventListener('paste', e => {

});
